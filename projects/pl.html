Inne Projekty

Tu znajduje się kompletna lista moich projektów oraz projektów, do których kontrybułuję!

Po więcej informacji sprawdź moją stronę GitHub.

StrayexOS
My own 32bit operating system for Intel CPUs, i386 and x86-64 compatible. Part of Strayex Project
Strayex Shell
Cross-platform Command Line Interface for Windows, Linux and Hobby OSes. Part of Strayex Proejct
Strayex Update Center
Windows Forms app to manage, update and customise StrayexOS virtual machines and repos. Part of Strayex Project
ChangeLog Generator
Windows Forms app to create changelogs more dynamically.
Reflex Trainer
Windows Forms app to train point reflex. Nice tool to train for FPSes.
MyWorldRP
Acronym for My World Role Play, Windows Forms half-graphic, half-text RPG game inspired by Sword Art Online.
