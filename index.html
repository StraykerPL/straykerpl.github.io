Project Strayex

Strayex is 32bit operating system for virtual machines. Created to be the Golden Mean between MikeOS and Linux.
There are two lines of development of Strayex: based on my own Strayex Kernel and based on Linux Kernel.
Here is the list of versions and download links:
Version Name Short Description Download
Strayex Legacy First version of Strayex, created with Bran's Kernel Development Tutorial. I made some modifications and addons. This version is only for playground. The code I created is too complicated and I needed to rewrite almost all from scratch. Click here!
Strayex Kernel v1.0.0 Alpha - Pretorian Distribution The kernel created after Strayex Legacy, ready to work with. Has basic funcions, can be run with any bootloader supporting Multiboot. It's great to have some fun with it, cos in this version kernel have basic kernel-space features. Click here!
Strayex Kernel v1.0.1 Alpha - Pretorian Distribution This version repairs some bugs from older one and add some improvments to keyboard and kernel space. This version is close to user space shell. Click here!
For complex and comfort use of Strayex, there are extra apps for ease out development and maintenance of project:
- Strayex Shell - CLI for Strayex, Windows, Linux and MacOS, Download here!
- Strayex Update Center - Windows Forms app for simple Strayex ISOs and repos management, Download here!
